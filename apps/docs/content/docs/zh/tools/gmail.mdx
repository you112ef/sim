[Gmail](https://gmail.com) 是 Google 的一项流行电子邮件服务，提供了一个强大的平台，用于发送、接收和管理电子邮件通信。Gmail 拥有超过 18 亿的全球活跃用户，提供了功能丰富的体验，包括强大的搜索功能、组织工具和集成选项。

使用 Gmail，您可以：

- **发送和接收电子邮件**：通过简洁直观的界面与联系人交流
- **组织邮件**：使用标签、文件夹和过滤器保持收件箱井井有条
- **高效搜索**：利用 Google 强大的搜索技术快速找到特定邮件
- **自动化工作流**：创建过滤器和规则以自动处理收到的邮件
- **随时随地访问**：通过设备同步的内容和设置使用 Gmail
- **与其他服务集成**：连接 Google 日历、云端硬盘和其他生产力工具

在 Sim 中，Gmail 集成使您的代理能够以编程方式发送、读取和搜索电子邮件。这为强大的自动化场景提供了可能，例如发送通知、处理收到的邮件、从电子邮件中提取信息以及管理通信工作流。您的代理可以撰写和发送个性化电子邮件，使用 Gmail 的查询语法搜索特定邮件，并从电子邮件中提取内容以用于工作流的其他部分。即将推出的功能还将允许代理实时监听新邮件，从而实现基于收到的邮件触发操作的响应式工作流。这一集成弥合了您的 AI 工作流与电子邮件通信之间的差距，实现了与全球最广泛使用的通信平台之一的无缝交互。

## 使用说明

将 Gmail 集成到工作流程中。可以发送、读取和搜索电子邮件。需要 OAuth 授权。可在触发模式下使用，当收到新邮件时触发工作流程。

## 工具

### `gmail_send`

使用 Gmail 发送电子邮件

#### 输入

| 参数 | 类型 | 必需 | 描述 |
| --------- | ---- | -------- | ----------- |
| `to` | 字符串 | 是 | 收件人电子邮件地址 |
| `subject` | 字符串 | 是 | 邮件主题 |
| `body` | 字符串 | 是 | 邮件正文内容 |
| `cc` | 字符串 | 否 | 抄送收件人（逗号分隔） |
| `bcc` | 字符串 | 否 | 密送收件人（逗号分隔） |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `content` | 字符串 | 成功消息 |
| `metadata` | 对象 | 邮件元数据 |

### `gmail_draft`

使用 Gmail 草拟电子邮件

#### 输入

| 参数 | 类型 | 必需 | 描述 |
| --------- | ---- | -------- | ----------- |
| `to` | 字符串 | 是 | 收件人电子邮件地址 |
| `subject` | 字符串 | 是 | 邮件主题 |
| `body` | 字符串 | 是 | 邮件正文内容 |
| `cc` | 字符串 | 否 | 抄送收件人（逗号分隔） |
| `bcc` | 字符串 | 否 | 密送收件人（逗号分隔） |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `content` | 字符串 | 成功消息 |
| `metadata` | 对象 | 草稿元数据 |

### `gmail_read`

从 Gmail 读取电子邮件

#### 输入

| 参数 | 类型 | 必需 | 描述 |
| --------- | ---- | -------- | ----------- |
| `messageId` | string | 否 | 要读取的邮件 ID |
| `folder` | string | 否 | 要从中读取电子邮件的文件夹/标签 |
| `unreadOnly` | boolean | 否 | 仅检索未读邮件 |
| `maxResults` | number | 否 | 要检索的邮件最大数量 \(默认值: 1，最大值: 10\) |
| `includeAttachments` | boolean | 否 | 下载并包含电子邮件附件 |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `content` | string | 电子邮件的文本内容 |
| `metadata` | json | 电子邮件的元数据 |
| `attachments` | file[] | 电子邮件的附件 |

### `gmail_search`

在 Gmail 中搜索电子邮件

#### 输入

| 参数 | 类型 | 必需 | 描述 |
| --------- | ---- | -------- | ----------- |
| `query` | string | 是 | 邮件搜索查询 |
| `maxResults` | number | 否 | 要返回的结果最大数量 |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `content` | string | 搜索结果摘要 |
| `metadata` | object | 搜索元数据 |
