接続は、ワークフロー内のブロック間でデータが流れるための経路です。情報があるブロックから別のブロックへどのように渡されるかを定義し、複雑な多段階プロセスを作成することができます。

<Callout type="info">
  適切に設定された接続は、効果的なワークフローを作成するために不可欠です。接続によって、
  データがシステム内をどのように移動し、ブロック同士がどのように相互作用するかが決まります。
</Callout>
