## チャット変数の使用

チャットモードでは、特別な変数を通じてユーザー入力と会話コンテキストにアクセスできます：

- **`<start.input>`** - ユーザーのメッセージテキストが含まれます
- **`<start.conversationId>`** - 会話スレッドの一意の識別子
- **`<start.files>`** - ユーザーがアップロードしたファイルの配列（ある場合）
